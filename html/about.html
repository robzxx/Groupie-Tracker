Ce site, vous permettra de retrouver vos artistes favoris à l'aide d'un système de filtres et de barre de recherche afin de trouver les dates de leurs prochains concerts ainsi qu'une multitude d'informations les concernant


Groupie tracker a pour projet de récupérer et de traiter les données d'une API sur le thème d'un site filtrant une multitude de groupes d'artistes ainsi que leurs dates de prochains concerts dans le monde les langagues de programmations autorisé sont le GO, HTML / CSS , JSON JS
